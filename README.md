## 📌 Sobre a Disciplina

A disciplina **Plataformas de Prototipação para Internet das Coisas** tem como objetivo capacitar o estudante a analisar, selecionar e utilizar diferentes plataformas de hardware e software aplicadas à prototipação de soluções IoT.

São exploradas arquiteturas baseadas em microcontroladores (MCU) e microprocessadores (MPU), com foco prático em ESP32, além da análise comparativa de plataformas como:

- Arduino  
- NodeMCU  
- Raspberry Pi  
- Orange Pi  

Ao final da disciplina, o aluno deverá ser capaz de:

- Compreender arquiteturas embarcadas aplicadas à IoT  
- Selecionar plataformas adequadas conforme requisitos técnicos  
- Integrar hardware, firmware e computação em nuvem  
- Projetar e prototipar uma solução IoT completa
